A.4 Palette anti-patterns · don'ts

  • Cycle-pink / Flo-magenta в primary brand — мгновенный «teen-period-tracker» bucket. Запрещено.
  • Real acne / hirsutism close-ups в onboarding — aversive + compliance dignity. Illustrations only до disclaimer-accept.
  • Red / yellow severity stoplight — clinical-cold, kills warmth. Use sage-warm → rose-gold → deep rose-gold вместо.
  • Anti-Metformin / anti-BC rhetoric, даже для Maya. Neutral framing only.
  • Diagnosis-language — никогда «you have PCOS». Always «your AMH 5.8 maps to PCOS-typical range per ESHRE 2023».
  • «Cure» / «heal» hormonal imbalance в copy. Always «manage / track / understand».
  • Definitive fertility predictions — TTC mode даёт window, не certainty.
  • Pure white / NHS-blue — pearl warmth = wedge against Flo / Clue gaslighting.
  • Cute period-tracker иллюстрации (uterus с глазами, dancing ovaries) — credibility-killer для Anna / Tanya.
  • Email-required onboarding — privacy-first. Sign-in with Apple optional, не required.
  • Cloud-by-default photo sync — local storage; iCloud opt-in only.
  • Identifiable minor faces на marketing / share-cards.
  • Gamification (XP, levels, confetti, «PCOS warrior!»). Adherence streak — единственное исключение.
  • AI Coach answers без RAG citation — каждый ответ цитирует ACOG / ESHRE / AAP / Endocrine Society / PubMed. No-citation = refusal pivot.

Locked from COMPLIANCE.md + Cysta_AI_PCOS_Master_v2_FINAL.md PART 6. Each "won't build" item is consilium-locked unless founder explicitly overrides in DECISIONS.md.

  • ❌ won't build Generic period tracker — Flo / Clue own it; PCOS women are gaslit by them.
  • ❌ won't build Calorie counter — MyFitnessPal wins; PCOS needs glycemic, not calorie.
  • ❌ won't build Generic fitness — Apple Fitness+ is cheaper; PCOS needs HIIT + resistance specifics, not yoga playlists.
  • ❌ won't build Cute period illustrations (uterus-with-eyes, dancing ovaries) — credibility-killer for Anna / Tanya.
  • ❌ won't build Email-required onboarding — privacy-first; Sign-in-with-Apple optional.
  • ❌ won't build Cloud-by-default photo sync — local storage, iCloud opt-in.
  • ❌ won't build Identifiable minor faces in marketing — compliance hard-line.
  • ❌ won't build Gamification (XP, levels, confetti, "PCOS warrior!") — except a single adherence streak.
  • ❌ won't build Community feed Y1 — Cysterhood owns this; we focus AI authority instead.
  • ❌ won't build Definitive fertility predictions ("you'll conceive on X day") — TTC gives window, not certainty.
  • ❌ won't build Anti-Metformin / anti-BC rhetoric — Maya tempts this; refuse, neutral framing per COMPLIANCE.md.
  • ❌ won't build "Cure PCOS" copy anywhere — compliance hard-line. Always "manage / track / understand".
  • ❌ won't build Diagnosis claims ("you have PCOS") — Type Quiz is self-assessment educational, never diagnostic.
  • ❌ won't build AI Coach answers without RAG citations — every answer cites ACOG / ESHRE / AAP / Endocrine Society / PubMed or refuses.
  • ❌ won't build Symptom checker — WebMD is free; PCOS needs chronic management, not acute triage.
